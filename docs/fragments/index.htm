About Luga Router

Luga Router is a feature-rich JavaScript library for client-side routing. It's compact, focused and extensible

Our mission

While developing Luga Router we had a few goals in mind:

  • Do not accept a regular expression as path
  • The router is not a singleton, you may run multiple instances of it
  • Make everything extensible and open to customization

Unit tests

Luga Router contains a comprehensive set of unit tests, developed with Jasmine

The unit tests play a fundamental role in ensuring the code is robust and, at the very same time, provide plenty of useful information about the API

Code coverage is usually around 100%. You can check coverage report by yourself

Dependencies and compatibility

Luga Router depends on Luga JS. To satisfy the dependency only luga.common.min.js is required

Browser compatibility

Luga Router is designed to work in IE11 and most recent versions of Chrome, FireFox, Edge and Safari

Bower

You can install Luga Router using bower by running: bower install luga-router
